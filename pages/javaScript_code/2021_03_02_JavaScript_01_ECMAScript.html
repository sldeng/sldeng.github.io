    # ECMAScript

    ECMAScript,即ECMA-262定义的语言，并不局限于web浏览器，事实上，这门语言没有输入和输出之类的方法，ECMA-262将这门语言作为一个基准来定义，以便在它之上在构建更稳定的脚本语言。web浏览器只是ECMAScript实现可能存在的一种宿主环境。宿主环境提供ECMAScript的基准实现和与环境自身交互必须的扩展。扩展使用ECMAScript核心类型和语法，提供特定于环境的额外功能，其他宿主环境还有服务器端JavaScript平台Nodejs和即将被淘汰的Adobe Flash.
